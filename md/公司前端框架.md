# 前端框架

##### 	左侧选择树

```vue
<template #slider>
      <g-select-tree
          label="行业"
          :data="industryList"
          v-model="industryId"
      >
        <template #default="{data}">
          <div style="display: flex;justify-content: center">
			//可以设置图标
            <el-image :src="data.logo" style="width:30px;height: 30px;margin-right: 5px"></el-image>
            <div style="font-size: 18px;text-align: center">{{ data.name }}</div>
          </div>
        </template>
      </g-select-tree>
    </template>
```

##### 跟随主题变换字体颜色

```vue
<span class="dialog-font" :style="{color:$theme == 'dark'?'#fff':'#000'}">{{ content }}</span>
```

##### 列表自定义按钮

```vue
<g-table-column v-if="isHasSomePermission" label="操作" width="300px" fixed="right">
        <template #default="{ row }">
          <udOperation :data="row" url="/admin/sourceDirectory/" :permission="permission">
            <template #rightArea>
              <el-button :disabled="row.url === ''" type="primary" @click="download(row)">下载</el-button>
            </template>
          </udOperation>
        </template>
      </g-table-column>
```
